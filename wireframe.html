What is Spring AI and how does it differ from LangChain4j?
Claude
Spring AI is a framework that brings AI capabilities natively into the Spring ecosystem. It provides a consistent abstraction layer over multiple AI providers — Anthropic, OpenAI, Ollama, and others — through a unified ChatClient API.

Key differences from LangChain4j:
  • Spring AI integrates directly with Spring Boot autoconfiguration and dependency injection.
  • It uses Spring's advisor pattern for cross-cutting concerns (RAG, memory, logging).
  • LangChain4j is framework-agnostic; Spring AI is opinionated about the Spring context.

Explain retrieval-augmented generation in 3 sentences.
Claude · streaming
Retrieval-Augmented Generation (RAG) enhances a language model by first retrieving relevant passages from an external knowledge base — using semantic similarity search over a vector store — and then injecting those passages into the model's prompt as grounding context. This allows the model to answer questions about private or up-to-date information it was not trained on, reducing hallucination by anchoring responses in retrieved facts. The retrieval step uses embedding models to convert both the query and stored documents into high-dimensional vectors, measuring closeness via cosine similarity
